Authentication Flow

Getting started with Call5 Democracy

Landing Page (Pre-Login)

  • "Sign up with Email" button for local authentication (v0.4.03)
  • Voter question form accessible to all visitors (anonymous or authenticated)
  • Mobile-responsive design with professional branding

Dashboard

Your central hub for network management and civic engagement

Main Dashboard
The command center for your Call5 network. View recruitment statistics, share your unique invite link, and access all platform features from a single, organized interface.
  • Welcome card with personalized greeting and recruit count (3 people recruited)
  • Invite link section with unique referral URL and copy-to-clipboard functionality
  • Quick actions: Share invite, message recruiter/recruits, view messages
  • Group management: Create new groups, view your groups
  • Platform tools: Submit voter questions, admin access, network tree
  • Mobile-optimized with vertically stacked buttons for easy thumb navigation
The Call5 Model
Each member recruits 5 people, who each recruit 5 more. Your unique invite link tracks your network: 1 → 5 → 25 → 125 → 625 → 3,125. This exponential growth creates sustained civic engagement beyond traditional election cycles.

Group Management

Organize your network into civic action groups

Create a Group
Start organizing your recruits into focused civic groups. Groups provide structure for collective action, whether organized by neighborhood, issue, or shared interest.
  • Group Name (required) - Examples: "54 Ave Group", "RAI Network", "Tech Volunteers"
  • Description (optional) - Explain the group's purpose or focus area
  • Only users with recruit_count > 0 can create groups
  • Group founder automatically becomes group admin
  • Can invite direct recruits to join after creation
Group Creation Community Building
Your Groups
View all groups you've created or joined. Each group card shows membership details, your role, and when it was created.
  • Role badges: Blue "Founder" badge for groups you created, "Member" for groups you joined
  • Member count displayed for each group (e.g., "3 members")
  • Creation info: See who founded each group and when
  • Click group name to access management page with full member list
  • Groups shown: "54 Ave Group", "15th Ave NW", "See Out My Space group"
Groups vs. Recruitment
Your recruitment network (who invited whom) is separate from group membership. You recruit individuals through invite links, then organize them into named groups for coordinated action. Members can belong to multiple groups simultaneously.

Messaging System

Direct communication between recruiters and recruits

Compose Message
Send direct messages to people in your recruitment network. Recruiters can message any of their recruits; recruits can message their recruiter.
  • Recipient dropdown: Shows only valid messaging relationships (your recruits or your recruiter)
  • Subject line: Optional, defaults to "Message from [your name]"
  • Message body: Required, supports 1-2000 characters
  • Email notification: Recipient receives email via MailGun with link to reply
  • Access control prevents messaging arbitrary users
1:1 Messaging Network Communication
Messages Inbox
View all your conversations with recruiters and recruits. Each conversation shows a preview of the most recent message and timestamp.
  • Conversation cards for each messaging relationship
  • Message preview: First ~100 characters of most recent message
  • Timestamp: Shows when last message was sent
  • Quick compose: "New Message" button opens compose form
  • Click conversation to view full message thread
  • Conversations shown with Max Girouard, Jeff Jordan, David Fallon
Future: v0.4.02 Enhancement
The next version will add full conversation threading, unread message counts, read receipts, and the ability to message directly from the network tree visualization. Email notifications will include message previews and one-click reply links.

Administrative Tools

Platform insights and network analytics

Comprehensive user report for platform administrators. Only accessible to users with admin privileges (currently Tony Byorick and Max Girouard).
  • Registered Users table: All authenticated users with ID, name, email, and unique invite code
  • Anonymous Submissions table: Unregistered users who submitted voter questions
  • Current users shown: Alex Carter, Jeff Jordan, Irene Ferrante, David Fallon, TB Bryan, Max, Tony
  • Track platform growth and identify potential recruits
  • Verify referral tracking is working correctly
  • Data for strategic outreach decisions

Network Tree
Visual representation of the Call5 recruitment network showing parent-child relationships and network depth.
  • Hierarchical display: Tony Byorick at root with 3 direct recruits
  • Recruit counts: Each name shows number of people they've recruited (in parentheses)
  • Network chains visible: Tony → David Fallon → (David's recruit), Tony → TB Bryan → Alex Carter

  • Future: Interactive visualization with collapsible branches and search
Network Growth Tracking
The network tree helps identify top recruiters, visualize growth patterns, and verify that the referral tracking system is working correctly. Each branch represents a recruitment chain, showing how civic engagement spreads through the network.
